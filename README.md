# Aiko Teste

Este projeto foi desenvolvido com base no desafio proposto pela Aiko para o processo seletivo de estágio:

https://github.com/aikodigital/teste-backend-estagio-v3

Além de implementar os requisitos solicitados no desafio, também desenvolvi uma interface web para proporcionar uma experiência de uso mais intuitiva e acessível, permitindo que os usuários interajam com o sistema de forma simples e eficiente.

## Tecnologias Utilizadas

### Backend
- Java
- Spring Boot
- JWT (JSON Web Token)
- PostgreSQL

### Frontend
- Next.js
- React
- Tailwind CSS

## Funcionalidades

- Cadastro de usuários
- Autenticação de usuários (login e logout)
- Cadastro de equipamentos
- Exclusão de equipamentos
- Monitoramento do estado dos equipamentos
- Monitoramento da localização dos equipamentos
- Visualização de gráficos de lucro por equipamento

## Objetivo

O objetivo deste projeto foi colocar em prática conceitos de desenvolvimento full stack, incluindo a criação de APIs REST, autenticação com JWT, integração com banco de dados PostgreSQL e desenvolvimento de interfaces modernas utilizando React e Next.js.
